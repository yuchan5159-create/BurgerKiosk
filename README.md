#(C# 코딩) BurgrKiosk

## 개요
-C# 프로그래밍 학습
-1줄 소개: 햄버거 주문을 위한 키오스크
-사용한 플랫폼:
 -C#, .NET Windows Forms, Visual Studio, GitHub
-사용한 컨트롤:
 -Label, button, RadioButton, checkBox, pictureBox, GroupBox
-사용한 기술과 구현한 기능:
 -Visual Studio를 이용하여 UI디자인
 -IF 조건절을 이용해 가격 업데이트

## 실행 화면
-코드의 실행 스크린샷과 구현 내용 설명
